# LoraDataViz

[ESP8266](https://docs.platformio.org/en/latest/boards/espressif8266/nodemcuv2.html) based Morse code/serial data transceiver and education tool.

Final Semester Project (FSP) of students of Tutorial Class E2T2 2025/2026, Kolej Matrikulasi Kejuruteraan Kedah (KMKK) under the guidance of Mr. Ahmad Hazeri, Electrical & Electronics Engineering Lecturer.

# Mac address

- Unit 1: `68:C6:3A:B2:F9:20`
  {0x68, 0xC6, 0x3A, 0xB2, 0xF9, 0x20}

- Unit 2: `84:F3:EB:D5:7A:92`
  {0x84, 0xF3, 0xEB, 0xD5, 0x7A, 0x92}
